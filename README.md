  # GBSW-09
캡스톤

## 프로젝트 개요 
학교 학생들이 필요한 물품을 필요한 만큼 구매하기 어려운 상황에서, 대량 구매를 요구하는 경우가 많아 이를 해결하고자 기획.

## 사용 기능
1. 한 사용자가 공동구매를 위한 게시물을 작성.
2. 다른 사용자가 게시물에 관심을 가지고 채팅 신청을 통해 의견을 조율.
3. 서로 의견을 맞춘 뒤 물품을 공동 주문.
